This assignment aims to help you practice the hash tables, especially collision resolution with open addressing in hash tables. 
Your main task in this assignment is to develop a student attendance list for a seminar using C programming language.

Overview:
The office of international student affairs is hosting a seminar to acquaint newcomers with the rules and regulations and the different facilities METU NCC has to offer. 
However, due to Covid restrictions, the office intends to make a "signup to attend" application to know the number of attending students in advance so a suitable 
venue can be picked. In this assignment, you will develop said application.

Example input is on the main part of the assignment.
